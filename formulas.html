<div class="eq-c">

</div>

.dblarrow {
    font-size: 125%;
    top: -0.4ex;
    margin: 0 2px;
}
.dblarrow:after {
    content:"\2190";
    position: absolute;
    left: 0;
    top: 0.5ex;
}

c(NaOH) =
n(NaOH) / V(NaOH)
= 1.2345 mol dm-3

HTML Código:

<div class="eq-c">
<i>c</i>(NaOH) =
<div class="fraction">
<span class="fup"><i>n</i>(NaOH)</span>
<span class="bar">/</span>
<span class="fdn"><i>V</i>(NaOH)</span>
</div>
= 1.2345 mol dm<sup>-3</sup>
</div>

CSS Código:

.fraction {
    display: inline-block;
    vertical-align: middle; 
    margin: 0 0.2em 0.4ex;
    text-align: center;
}
.fraction > span {
    display: block;
    padding-top: 0.15em;
}
.fraction span.fdn {border-top: thin solid black;}
.fraction span.bar {display: none;}

<span class="lim"><i>n</i>=0</span>
</span>

<span class="fup">(<i>x</i> + 1)<sup><i>n</i></sup></span>
<span class="bar">/</span>
<span class="fdn"><i>n</i></span>
</div>
</div>


<div class="eq-c">
<span class="intsuma">
<span class="lim-up">2&pi;</span>
<span class="sum">&int;</span>
<span class="lim">0</span>
</span>
sin<i>x</i> d<i>x</i> = 0
</div>

Qué es lo que en caja de CSS

El modelo de caja CSS es esencialmente una caja que se envuelve alrededor de los elementos HTML, y se compone de: márgenes, bordes, el relleno y el contenido real.
